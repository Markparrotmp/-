# 🏐 Волейбольный бот для Телеграма

Бот присылает в беседу опрос **каждый день в 12:00 по Москве** с тремя вариантами:

- Да ✅
- Нет ❌
- 50 на 50 🤔

Голосование не анонимное — видно, кто и как проголосовал.

---

## Что понадобится

- Компьютер или сервер, где Python 3.9+ и есть возможность держать процесс запущенным (бот работает, только пока запущен).
- Аккаунт в Телеграме.

## Шаг 1. Создать бота и получить токен

1. Открой в Телеграме [@BotFather](https://t.me/BotFather).
2. Отправь `/newbot`, придумай имя и username.
3. BotFather пришлёт **токен** вида `123456789:AAE...`. Сохрани его.

## Шаг 2. Добавить бота в беседу

1. Добавь своего бота в волейбольную беседу.
2. Дай ему право отправлять сообщения (в группах по умолчанию есть).
3. Чтобы бот видел команды в группе, у @BotFather можно отключить Group Privacy
   (`/setprivacy` → выбрать бота → `Disable`). Для отправки опросов это не обязательно,
   но команды `/chatid` и `/poll` в группе будут работать надёжнее.

## Шаг 3. Установить зависимости

```bash
pip install -r requirements.txt
```

## Шаг 4. Узнать ID беседы

Токен уже есть, ID чата — ещё нет. Узнаём его так:

```bash
export BOT_TOKEN='ВАШ_ТОКЕН'   # Windows PowerShell: $env:BOT_TOKEN='ВАШ_ТОКЕН'
python bot.py
```

Бот запустится. Теперь в беседе отправь команду `/chatid` — бот ответит числом
(для групп обычно с минусом, например `-1001234567890`). Скопируй его и останови
бота (`Ctrl+C`).

## Шаг 5. Запустить бота по-настоящему

Задай обе переменные и запусти:

```bash
export BOT_TOKEN='ВАШ_ТОКЕН'
export CHAT_ID='-1001234567890'
python bot.py
```

Готово. Пока процесс работает, каждый день в 12:00 по Москве в беседу будет
приходить опрос.

> Можно скопировать `.env.example` в `.env`, вписать туда значения и загружать их
> удобным способом (например, через `python-dotenv` или `export $(cat .env | xargs)`).

## Команды бота

| Команда   | Что делает                                               |
|-----------|----------------------------------------------------------|
| `/start`  | Приветствие и справка                                    |
| `/chatid` | Показывает ID текущего чата (нужно для настройки)         |
| `/poll`   | Отправляет опрос прямо сейчас — удобно для проверки       |

## Как поменять время или текст

Всё в начале файла `bot.py`:

```python
QUESTION = "Играем сегодня в волейбол? 🏐"
OPTIONS = ["Да ✅", "Нет ❌", "50 на 50 🤔"]
TIMEZONE = ZoneInfo("Europe/Moscow")
POLL_TIME = time(hour=12, minute=0, tzinfo=TIMEZONE)
```

Поменяй `hour`/`minute` — и перезапусти бота.

## Чтобы бот работал без твоего компьютера

Держать процесс запущенным круглосуточно удобно на дешёвом VPS. Варианты:

- **systemd** (Linux-сервер) — создать сервис, который автоматически перезапускает бота.
- **screen / tmux** — простой способ оставить процесс работать после выхода из SSH.
- Любой хостинг, где можно запустить постоянный Python-процесс.
